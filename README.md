O **TaskBoard** é uma aplicação web completa com banco de dados integrado, desenvolvida em Python. O objetivo deste projeto é demonstrar o domínio sobre o ciclo de vida completo da manipulação de dados (CRUD) em um ambiente corporativo.

---

## 🚀 Funcionalidades (CRUD Implementado)
* **Create (Criar):** Inserção de novas tarefas com status inicial "Pendente".
* **Read (Ler):** Consulta em tempo real ao banco de dados e exibição estruturada utilizando Pandas.
* **Update (Atualizar):** Atualização do status de tarefas específicas para "Concluído" via interface.
* **Delete (Excluir):** Remoção definitiva de registros do banco de dados.

## 🛠️ Tecnologias Utilizadas
| Tecnologia | Função |
| :--- | :--- |
| **Python** | Lógica de programação e backend |
| **SQLite3** | Banco de dados relacional embarcado |
| **Pandas** | Leitura e renderização dos dados via SQL (DataFrames) |
| **Streamlit** | Construção da interface de usuário (UI) |

## 💡 Impacto do Projeto
Este sistema prova a capacidade de conectar aplicações web a bancos de dados relacionais, utilizando comandos SQL (`INSERT`, `SELECT`, `UPDATE`, `DELETE`) de forma segura, com tratamento de estado na interface e foco na experiência do usuário.
